    Rich Picture

    Introdução

    Um Rich Picture é uma maneira de explorar, reconhecer, definir uma situação e expressar pensamentos através de diagramas para criar um modelo mental preliminar. Um Rich Picture ajuda a abrir uma discurssão e chegar a um entendimento geral de uma situação. E a abordagem adotada pela equipe foi na elaboração de cada parte dos interressados para chegamos ao entendimento geral do produto Triagil.
